# 🩺 DiaCare AI

## AI-Powered Diabetes Risk Prediction and Health Assessment Web Application

DiaCare AI is a smart healthcare web application that helps users assess their diabetes risk using symptoms, lifestyle habits, and medical parameters. The platform provides AI-powered health insights, personalized recommendations, downloadable PDF reports, email report delivery, and multilingual support.

---

## 📌 Project Overview

Diabetes is a rapidly growing health concern worldwide. Early detection can help prevent severe complications and improve quality of life.

DiaCare AI aims to provide an easy-to-use platform where users can assess their diabetes risk and receive health recommendations based on their inputs.

---

## 🎯 Objectives

- Predict diabetes risk using health-related information.
- Increase awareness about diabetes and preventive healthcare.
- Provide personalized recommendations for users.
- Generate downloadable PDF health reports.
- Support multiple languages for accessibility.
- Offer AI-powered healthcare assistance.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/diacare-ai.git
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
